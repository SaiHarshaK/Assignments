1) Create a file "inp-params.txt". Here, input given will be parsed by the program
    Format to be followed is :

    n - size (power of 10)
    m - number of threads

    Example:
    8 10


2)
    a) Compile the program file by executing following command:
    g++ Src-CS17BTECH11036.cpp -lpthread -o program


3)
    a) Run the executable file by executing :
    ./program

4) Check “Primes-DAM.txt”, “Primes-SAM1.txt” and “Primes-SAM2.txt” for Prime numbers generated by the respectice method
    Check "Times.txt" for time taken by DAM, SAM1 and SAM2 respectively.
